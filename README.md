# FPGA並行編程

> -- 以HLS實現信號處理為例

國內鮮有介紹HLS的書，我們希望通過翻譯`Parallel Programming for FPGAs`這本書，讓更多的人來了解HLS和FPGA開發。

## 本書電子版pdf

> 歡迎關注公眾號`PYNQ中文社區`獲取最新版電子pdf
> 回覆 `pp4fpgas` 即可獲得
